In this guide, we will explain how to integrate Kazaplan on your website.

There is two type of integration:
 - **Basic** (very fast implementation): Kazaplan standalone within an iFrame, your user accounts are not linked with kazaplan account, all the authentication process (signin / signup) is  perform inside the widget. Very fast to implement. see [Basic Setup Guide](/docs/basic_setup)
